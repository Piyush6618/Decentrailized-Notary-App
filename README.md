# 🔐 BlockStamp - Decentralized Notary

A **privacy-first decentralized notarization platform** that allows users to securely hash, notarize, and verify documents on the blockchain.

---

## 🚀 Live Demo
👉 https://decentralized-notary-app.vercel.app

---

## 📌 Problem Statement

- 💰 **$40B+ lost annually** due to document fraud  
- 🌍 **1B+ people lack access** to reliable notarization  
- ⏳ Traditional systems are slow, centralized, and vulnerable  

---

## 💡 Solution

BlockStamp provides a **trustless notarization system** using blockchain:

- 🔐 Documents are hashed **client-side**
- ⛓️ Hash stored on blockchain (immutability)
- ✍️ Digital signatures ensure authenticity
- 🔍 Anyone can verify document integrity instantly

---

## 🛠️ Tech Stack

### Frontend
- React.js (Vite)
- Ethers.js
- Web Crypto API (SHA-256)

### Blockchain
- Solidity Smart Contract
- Polygon (Mumbai Testnet)

### Deployment
- Vercel

---

## ⚙️ Features

- 📄 Upload document & generate SHA-256 hash
- 🔗 Store hash on blockchain
- 🦊 MetaMask wallet integration
- ✅ Verify document authenticity
- ⚡ Fast, gas-efficient notarization

---

## 🧪 How It Works

1. Upload document  
2. Hash generated in browser (privacy-first)  
3. Connect wallet  
4. Store hash on blockchain  
5. Verify anytime using same document
